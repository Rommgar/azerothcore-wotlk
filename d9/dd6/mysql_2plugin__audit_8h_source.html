111 /*************************************************************************
112  Here we define the descriptor structure, that is referred from
113  st_mysql_plugin.
114 
115  release_thd() event occurs when the event class consumer is to be
116  disassociated from the specified THD. This would typically occur
117  before some operation which may require sleeping - such as when
118  waiting for the next query from the client.
119 
120  event_notify() is invoked whenever an event occurs which is of any
121  class for which the plugin has interest. The first word of the
122  mysql_event argument indicates the specific event class and the
123  remainder of the structure is as required for that class.
124 
125  class_mask is an array of bits used to indicate what event classes
126  that this plugin wants to receive.
127 */
