void my_thread_basic_global_reinit ( void  )

Re-initialize components initialized early with my_thread_basic_global_init. Some mutexes were initialized before the instrumentation. Destroy + create them again, now that the instrumentation is in place. This is safe, since this function() is called before creating new threads, so the mutexes are not in use.
