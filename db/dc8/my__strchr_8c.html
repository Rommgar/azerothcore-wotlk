Calculate the length of the initial segment of 'str' which consists entirely of characters not in 'reject'.

Note
The reject string points to single-byte characters so it is only possible to find the first occurrence of a single-byte character. Multi-byte characters in 'str' are treated as not matching any character in the reject string.
